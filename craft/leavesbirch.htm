Blocks -> Leaves (Birch)
Type: Block
MC ID: 18:2

Description

Leaves can be obtained using Shears on Leaf Blocks of a Tree. They grow when a Sapling turns into a Tree. Destroying Leaves can cause Saplings to drop.
