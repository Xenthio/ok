Blocks -> Fire
Type: Block
MC ID: 51

Description

Fire is created using Flint and Steel on a flamable block. It is also created by Lava igniting flamable blocks. Fire can spread to flamable blocks that are near to it. Mobs or players touching Fire will be set on Fire. Water can put Fire out.
