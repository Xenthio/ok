Crafting
Crafting is a way to turn basic materials (Such as Cobblestone) into Tools/Weapons/Etc. All players have a 2x2 crafting grid in their inventory which can be used at anytime. To access a 3x3 crafting grid you need to use a Workbench which then allows you to create more types of items. To use a Workbench you need to right click on it once placed.

To craft items you need to arrange the materials in the correct pattern for the item you wish to create. The slots in which you place the materials doesn't matter as long as the pattern is correct.

Below is a list of the current items which can be crafted and the patterns used to craft them. There are currently 153 craftable items.
