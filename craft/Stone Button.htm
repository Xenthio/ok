Stone Button
 
 
 
 
 
 
 
1

Type: Item
MC ID: 77

Description

Stone Buttons can be used to activate objects like Iron Doors and Redstone Wires. They activate objects for about 1-2 seconds. Stone Buttons can be pressed by Left Clicking on them.
