Introduction

Spawning items allows you to obtain items without mining/harvesting them. Every Item and Block has its own unique Item ID.

ID List

To spawn an item you must first know the Item ID that is assigned to the item. You can find a list of the Item ID's on the ID List Page. Certain Items/Blocks have a Secondary ID which is represented in Minecraft Info's ID List as Primary:Secondary.

Singleplayer

If you want to spawn items in Singleplayer you would need to use an Inventory Editor such as INVedit or a mod such as Single Player Commands. Note that Secondary ID's may not be supported by certain community made tools/mods.

Multiplayer

If you want to spawn items in Multiplayer then you need to find a server which allows it.

You can spawn on a vanilla server using:
/give (Username) (Primary ID) (Amount)

For example if you would like to give Notch 64 Sapling then it would look like:
/give Notch 6 64

If the server has ScrapBukkit installed then you spawn items using:
/give (Primary ID):(Seconday ID) (Amount) (Username)

For example if you would like to give Notch 64 Sapling (Pine) then it would look like:
/give 6:1 64 Notch

Note that Vanilla servers do not accept a Secondary ID.
